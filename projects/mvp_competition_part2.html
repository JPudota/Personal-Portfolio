Property Analysis - Sprint 2

Designed interactive reports and dashboards in SSRS and Power BI to analyze school demographics and property median values.

Power BI Dashboard: Property Median Value Insights

Project Overview

  • To develop interactive, filterable reports and dashboards using SSRS and Power BI, delivering actionable insights into property values and school metrics across NSW based on the engineered data model.
  • Tools Used: SQL Server | SSRS | Power BI
  • Techniques: Parameterized Reporting, Drill-through Navigation, Geographic Mapping, Snowflake Schema Integration, Interactive Filtering, Visual Storytelling, Export-Ready Report Design

What I Did

  • Created a parameterized SSRS table report with suburb and postcode filters.
  • Included school details, ICSEA scores, and average property values in SSRS.
  • Published the SSRS report to a web portal and tested Excel/PDF exports.
  • Built Power BI dashboards showing property trends by suburb, city, postcode.
  • Added slicers, drill-throughs, map visuals, and a hyperlink to open the SSRS report from Power BI.

Key Deliverables

  • SSRS report with dynamic filters for Suburb and Postcode.
  • Export-ready formatting (PDF, Excel) with headers, footers, and styling.
  • Power BI dashboard with drill-through, slicers, and map visuals.
  • Linked navigation from Power BI to SSRS for unified reporting.
  • Interactive visual storytelling using property median value and school datasets
